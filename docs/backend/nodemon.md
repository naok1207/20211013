# nodemon

ファイルの変更を監視し、サーバーを再起動するためのnpmライブラリ

**インストール**
```
yarn add -D nodemon
```


**実行**
```
nodemon <ファイル名>
```

**オプション(今回使用)**
1. `--legacy-watch` windows環境でもnodemonを有効化する
2. `--config` 設定ファイルの場所を任意で設定する
